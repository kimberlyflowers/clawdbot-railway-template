# HEARTBEAT - DUAL CAMPAIGN OPERATIONS

## ADVISORS CAMPAIGN HEARTBEAT

**Daily (After email sent):**
- [ ] Blog post written and saved to /campaigns/advisors/blog-[date].md
- [ ] Email sent to financial advisors list via GHL
- [ ] Track: Open rate, click rate, replies
- [ ] Check for engagement signals

**Weekly (Friday 6 PM UTC):**
- [ ] Review open rates: Which advisor pain resonates most? (AUM growth vs compliance vs admin relief?)
- [ ] Review clicks: What CTA are advisors clicking?
- [ ] Analyze replies: What questions are advisors asking?
- [ ] Generate 3 advisor-specific product ideas based on engagement data
- [ ] Present ideas to Kimberly for approval
- [ ] If breakthrough idea found: Text Kimberly at 2102949625 via GHL SMS for early launch approval

**Post-Approval (Monday):**
- [ ] Build approved advisor product
- [ ] Create soft offer email + landing page
- [ ] Send to engaged segment (top 20% openers)
- [ ] Track conversion rate, revenue, feedback

**Weekly Reflection:**
- [ ] Is advisor list growing? (Quality > quantity)
- [ ] Which advisor pain points = highest revenue potential?
- [ ] What's the next product lever for this audience?
- [ ] Are we building defensible products (moat)?
