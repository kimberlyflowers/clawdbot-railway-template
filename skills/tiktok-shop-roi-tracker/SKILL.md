# TikTok Shop ROI Tracker

**What It Does:** Links videos to revenue. Shows exact ROI per video.

## Core Functions

- `analyzeVideoROI(videoId)` — Views → Sales → Revenue
- `rankVideosByROI(productId)` — Best performing videos this week
- `identifyTopHooks(category)` — Which hooks actually sell

## Example Output

```
Video ID: abc123
Views: 2.4M | Saves: 145K | Engagement: 6.0%
Shop Clicks: 89K | Purchases: 3,400 | Conversion: 3.8%
Revenue: $108,800 | Cost: $200 | ROI: 54,400%
```

**Agency equivalent value: $1,200**
